* 게임 진행 방식
	- 횟수 제한 방식
	- level up -> 횟수 증가
	- item으로 횟수 증가

* Level관리
	- 게임 상에서 획득한 돈으로 level up?
	- 경험치 증가로 level up?

* 점수제
	- 1승에 해당하는 점수 부여
	- 연승시에 보너스 점수 부여( 미리 정의된 연승 점수 or 누적 점수에 대해 연승에 대한 비율 계산으로 추가 점수 )

* 게임 머니 획득
	- 5연승마다 룰렛 기회 제공 -> 룰렛을 통해서 획득
	- 1승마다 룰렛 코인 제공 -> 게임 완료 후 룰렛을 통해 획득 - 기본 머니 보장, 일정 확률로 젝팟~!(아이템 사용 유도)
